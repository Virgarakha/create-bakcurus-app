# Backurus Documentation Site

Static multilingual documentation for Backurus.

## Coverage

- installation
- quickstart
- routing
- controllers
- ORM and models
- migrations and schema builder
- validation
- authentication
- queue and scheduler
- WebSocket
- plugins and facades
- CLI reference

## Features

- English and Indonesian translations in `docs/lang/`
- language switcher with `localStorage`
- dark mode toggle
- responsive sidebar layout
- client-side search
- copyable code blocks
- lightweight syntax highlighting

## Important Files

- `docs/assets/js/i18n.js`
- `docs/assets/js/main.js`
- `docs/lang/en.json`
- `docs/lang/id.json`
- `docs/*.html`
